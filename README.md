# homebrew-wayu

Homebrew tap for [wayu](https://github.com/dvrd/wayu) — shell configuration management CLI.

## Install

```bash
brew tap dvrd/wayu
brew install wayu
wayu init
```

## What is wayu?

wayu manages your shell configuration — PATH entries, aliases, environment variables, completions, and plugins — with a zero-dependency CLI written in Odin.
